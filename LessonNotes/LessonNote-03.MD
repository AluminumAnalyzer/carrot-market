자동완성 기능을 통해 Tailwind의 기능을 빠르게 자동완성 할수 있습니다.
className에서 클래스네임을 입력시 해당 클래스네임의 실제 CSS를 미리 보기로 제공하는 한편 색상의 경우 해당 색상을 미리 보기를 제공합니다.
심지어

```
    <div className='flex grid'>
    ...
    </div>
```

입력시 flex와 grid의 충돌조차 인식해 오류 발생을 알려줍니다.
